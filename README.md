# 🌦️ Simple Weather App

A beginner-friendly weather web app built using HTML, CSS, and JavaScript.

## 🚀 Features
- Search weather by city name
- Displays:
  - Temperature
  - Weather condition
  - Wind speed
- Clean and simple UI

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeatherMap API
